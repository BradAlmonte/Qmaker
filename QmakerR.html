About Qmaker

Qmaker is a lightweight, client‑side QR code generator built with HTML, CSS, and vanilla JavaScript. It was designed with the goal of proviting a service many providers try to charge for, for free.

Key features:

  • Customizable QR code content (text, URLs, Wi‑Fi credentials, etc.)
  • Error correction level selection (L, M, Q, H)
  • Adjustable size slider for resolution control
  • One‑click PNG download
  • Responsive design with a themed UI
